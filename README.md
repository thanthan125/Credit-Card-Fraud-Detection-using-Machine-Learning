# Credit Card Fraud Detection using Machine Learning

This project builds a machine learning model to detect fraudulent financial transactions.

## 📌 Objective
- Simulate transaction dataset
- Handle class imbalance
- Train classification model
- Evaluate fraud detection performance

## 🧠 Concepts Used
- Classification Modeling
- Random Forest
- Confusion Matrix
- Precision, Recall, F1-score
- Imbalanced Data Handling

## 📊 Outputs
- Confusion Matrix
- Classification Report
- Fraud detection metrics
- Transaction distribution analysis

## 🛠 Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📁 Files
- `fraud_detection.ipynb`
- `fraud_data.csv`
- `requirements.txt`

## 📈 Key Insight
The model identifies fraudulent patterns in transaction behavior using supervised learning techniques.
